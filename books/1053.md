# 指针运算符

| 运算符 | 描述             | 实例                       |
| ------ | ---------------- | -------------------------- |
| &      | 返回变量的地址。 | &a; 将给出变量的实际地址。 |
| *      | 指向一个变量。   | *a; 将指向一个变量。       |

```c
#include <stdio.h>
int main()
{
   int a = 4;
   short b;
   double c;
   int* ptr;
   /* & 和 * 运算符实例 */
   ptr = &a;    /* 'ptr' 现在包含 'a' 的地址 */
   printf("a 的值是 %d\n", a);
   printf("*ptr 是 %d\n", *ptr);
}
/*
a 的值是 4
*ptr 是 4
*/

```
