# 条件运算符

| 运算符 | 描述       | 实例                                 |
| ------ | ---------- | ------------------------------------ |
| ? :    | 条件表达式 | 如果条件为真 ? 则值为 X : 否则值为 Y |

```c
#include <stdio.h>
 
int main()
{
   int a = 4;
   short b;
   double c;
   /* 三元运算符实例 */
   a = 10;
   b = (a == 1) ? 20: 30;
   printf( "b 的值是 %d\n", b );
 
   b = (a == 10) ? 20: 30;
   printf( "b 的值是 %d\n", b );
}
/*
b 的值是 30
b 的值是 20
*/
```
